# Portfolio Website

Standalone portfolio project built with Vite + React + TypeScript + Tailwind CSS.

## Prerequisites

- Node.js 20 LTS (or newer)
- npm

## Run locally

```sh
npm install
npm run dev
```

The dev server usually starts at `http://localhost:8080`.

## Build

```sh
npm run build
```

## Preview production build

```sh
npm run preview
```

## Deploy (GitHub Pages)

```sh
npm run deploy
```
